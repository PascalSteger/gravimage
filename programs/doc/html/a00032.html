Detailed Description

Functions related to projection and deprojection of density in spherical models.

Conventions: rho, r, Mr denote 3D density, 3D radius, M(<3D radius) Sig, R, MR denote 2D density, 2D radius, M(<2D radius) SUM denotes main method = summing INT denotes main method = integrating NORM denotes main method = renormalization
