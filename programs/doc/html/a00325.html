def montepython.initialise.initialise (   custom_command = '')

Initialisation routine.

This function recovers the input from the command line arguments, from :mod:parser_mp, the parameter files.

It then extracts the path of the used Monte Python code, and proceeds to initialise a :class:data instance, a cosmological code instance.

Parameters

custom_command: str allows for testing the code
